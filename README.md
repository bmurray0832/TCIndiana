# TCIndiana CRM

A nonprofit CRM for Teen Challenge Indiana — replacing Bloomerang with a
tool that tracks **both prospects and donors** through a sales-style
pipeline, supports multiple centers, and accepts online giving directly.

Built from the workflow in `nonprofit_crm_v2_final.xlsm`.

See [`docs/PLAN.md`](docs/PLAN.md) for the full plan: objective, data
model, views, online-giving design, Bloomerang import, and phased
rollout.

## Stack

Next.js (App Router) · TypeScript · Tailwind · Prisma + Postgres ·
Auth0 · Stripe · Resend
